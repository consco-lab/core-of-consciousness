# Core Of Consciousness

从理论到实践：构建具备意识特征行为的系统——意识核

## 项目状态

**当前阶段**：理论框架正在构建，同时正在尝试构建工程验证 Demo。

## 核心理念

**设计目标**：本项目不满足于解释意识，而是试图构建它——通过工程实践，设计一个能够产生特定高阶行为特征（意识行为）的系统。

**设计方法**：以哲学和心理学为指导，脑神经科学为蓝图，计算机技术和机器学习作为工具。
