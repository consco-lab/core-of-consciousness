## 🎯 项目目标

### 核心理念

本项目不满足于解释意识，而是试图构建它——通过工程实践，设计一个能够产生特定高阶行为特征（意识行为）的系统。

### 意识行为的操作化定义

我们采用操作主义立场，以可观测的行为特征作为意识判定依据。

---

## 🧠 理论框架

### 核心思想

> **意识不是神秘礼物，而是复杂系统在与动态环境的持续交互中，为了最大化适应能力，而自然涌现的高阶行为特征的系统能力。**

### 意识涌现的三个必要条件

1. **系统内部复杂度**：模块化、层级化、丰富的反馈回路
2. **系统与环境动态交互**：在“行动-反馈”循环中学习和适应
3. **自我-环境边界的形成**：在信息流中区分“自身状态”与“外部世界”
